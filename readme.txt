== Description ==

WooCommerce is the most popular wordpress plugin for ecommerce with a templated pages. Editing or changing its templated pages requires basic knowledge in web development.

WooLayout is a powerful extension for WooCommerce that allows you to fully customize the WooCommerce pages. Create and Design your WooCommerce Pages by adding, switching or removing woocommerce elements by dragging and dropping woocommerce elements in the field. its simply and easy.

== Installation ==

= Minimum Requirements =

* WordPress 3.8 or greater
* Woocommerce Plugin

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of Woolayout, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “Woolayout” and click Search Plugins. Once you’ve found our plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
